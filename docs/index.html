Browser-Use Documentation

Overview

Browser-Use is a Python library and API that enables AI systems to interact with web browsers. It provides a window to the internet for AI agents, allowing them to automate web browsing tasks, gather information, and interact with web applications.

Installation

Python Package

pip install browser-use

Web UI

The Web UI can be installed separately:

cd web-ui
pip install -r requirements.txt

Configuration

Create a .env file based on the .env.example with your API keys for OpenAI and Anthropic:

OPENAI_API_KEY=your-openai-api-key
ANTHROPIC_API_KEY=your-anthropic-api-key

# Set to false to disable anonymized telemetry
ANONYMIZED_TELEMETRY=true

# LogLevel: Set to debug to enable verbose logging, set to result to get results only
BROWSER_USE_LOGGING_LEVEL=info

Basic Usage

Here's a simple example of using Browser-Use:

from browser_use import BrowserUse

browser = BrowserUse()
result = browser.browse(url="https://example.com", task="Find the main heading and any links")
print(result)

Web UI

To run the web UI:

cd web-ui
python webui.py

The UI will be available at http://localhost:7860 by default.
