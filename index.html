Browser-Use

A powerful web browser automation tool for AI agents. Browser-Use allows AI systems to interact with web browsers, providing them with a window to the internet.

Features

  • Web browser automation through Python and API interfaces
  • Support for interactive browsing sessions
  • Integration with OpenAI and Anthropic APIs
  • Customizable web UI for direct interaction

Documentation

For full documentation, installation instructions, and API reference, please visit the GitHub repository.

Web UI

The project includes a web-based user interface for direct interaction with the browser automation capabilities.

Getting Started

pip install browser-use
